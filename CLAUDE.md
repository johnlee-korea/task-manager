# 업무 일정 관리 웹앱 (Task Manager)

## 프로젝트 개요

개인 업무 일정을 관리하는 웹 애플리케이션

## 핵심 기능

- 일정 등록/수정/삭제 (CRUD)
- 캘린더 뷰 (월간/주간/일간)
- 우선순위 표시 (높음/중간/낮음)
- 알림/리마인더 기능

## 작업 규칙

1. **기획 단계**: 먼저 기능 목록과 구조를 정리하고, 승인 후 개발 시작
2. **개발 단계**: 코드에 한국어 주석을 달고, 파일별 역할을 명확히 분리
3. **검수 단계**: 코드 작성 후 버그 체크와 개선점을 리포트
4. **문서화 단계**: README와 사용법을 정리

## 기술 스택

- HTML, CSS, JavaScript (순수 웹 기술로 시작)
- 추후 필요시 React 전환 가능
