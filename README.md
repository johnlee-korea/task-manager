# Task Manager - 업무 일정 관리 웹앱

개인 업무 일정을 관리하는 웹 애플리케이션입니다.
별도 서버 없이 브라우저만으로 동작하며, 데이터는 LocalStorage에 저장됩니다.

## 기능 목록

### 로그인 / 비밀번호 보호
- 최초 접속 시 비밀번호 설정 화면 표시 (4자 이상)
- 이후 접속 시 비밀번호 입력 후 앱 진입
- 비밀번호 재설정 기능 (로그인 화면 하단)
- 비밀번호는 해시 처리되어 LocalStorage에 저장

### 일정 관리 (CRUD)
- 일정 등록: 제목, 날짜, 시작/종료 시간, 우선순위, 알림, 메모 입력
- 일정 수정: 기존 일정 클릭 후 내용 편집
- 일정 삭제: 삭제 확인 다이얼로그를 통한 안전한 삭제
- 자동 저장: 변경 시 LocalStorage에 즉시 반영

### 캘린더 뷰
- **월간 뷰**: 한 달 전체를 격자로 표시, 각 셀에 일정 카드 나열
- **주간 뷰**: 7일 타임라인 (06:00~23:00), 시간대별 일정 블록 표시
- **일간 뷰**: 하루 타임라인, 시간대별 상세 일정 표시
- 이전/다음 네비게이션, 오늘 버튼으로 빠른 이동
- 사이드바 미니 캘린더로 날짜 빠른 선택

### 우선순위 시스템
- 3단계 우선순위: 높음(빨강) / 중간(노랑) / 낮음(초록)
- 일정 카드에 색상으로 시각적 구분
- 사이드바 필터로 우선순위별 표시/숨김 전환

### 알림/리마인더
- 브라우저 데스크탑 알림 (Notification API)
- 앱 내 토스트 알림 (우측 상단 팝업)
- 알림 시점 선택: 정시, 10분 전, 30분 전, 1시간 전
- 30초 간격 자동 체크, 중복 알림 방지

## 실행 방법

### 바로 실행
별도 설치 없이 `index.html` 파일을 브라우저에서 열면 됩니다.

```
1. 프로젝트 폴더로 이동
2. index.html 파일을 더블 클릭하거나 브라우저에 드래그
3. 최초 실행 시 비밀번호를 설정하고, 이후에는 비밀번호를 입력하여 로그인
```

### 로컬 서버로 실행 (선택)

```bash
# Python 3
python -m http.server 8080

# Node.js (npx)
npx serve .

# VS Code
# Live Server 확장 설치 후 index.html에서 "Open with Live Server"
```

브라우저에서 `http://localhost:8080` 접속

### 지원 브라우저
- Chrome, Edge, Firefox, Safari 최신 2개 버전
- IE는 미지원

## 사용법

### 로그인
1. 최초 접속 시 비밀번호(4자 이상)를 설정
2. 이후 접속 시 설정한 비밀번호를 입력하여 로그인
3. 비밀번호를 잊은 경우 **"비밀번호 재설정"** 클릭 (기존 비밀번호 삭제 후 재설정)

### 일정 등록
1. 우측 상단 **"+ 새 일정"** 버튼 클릭, 또는 캘린더 빈 영역 클릭
2. 제목(필수)과 날짜(필수)를 입력
3. 필요시 시작/종료 시간, 우선순위, 알림, 메모를 설정
4. **"저장"** 버튼 클릭

### 일정 수정
1. 캘린더에서 일정 카드를 클릭
2. 모달에서 내용 수정 후 **"저장"** 클릭

### 일정 삭제
1. 일정 카드 클릭하여 수정 모달 열기
2. 좌측 하단 **"삭제"** 버튼 클릭
3. 확인 다이얼로그에서 **"삭제"** 클릭

### 뷰 전환
- 상단 중앙의 **월간 / 주간 / 일간** 탭 클릭
- **◀ ▶** 버튼으로 이전/다음 이동
- **"오늘"** 버튼으로 현재 날짜 복귀

### 우선순위 필터
- 좌측 사이드바의 체크박스로 우선순위별 표시 on/off
- 체크 해제 시 해당 우선순위 일정이 캘린더에서 숨겨짐

### 알림 설정
1. 일정 등록/수정 시 **"알림"** 드롭다운에서 시점 선택
2. 최초 실행 시 브라우저 알림 권한을 **"허용"** 해야 데스크탑 알림 수신 가능
3. 권한을 거부해도 앱 내 토스트 알림은 정상 동작

### 단축키
| 키 | 동작 |
|----|------|
| `ESC` | 열려있는 모달/다이얼로그 닫기 |

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - CSS Grid, Custom Properties, 애니메이션
- **JavaScript (ES5+)** - IIFE 모듈 패턴, LocalStorage API, Notification API
- 외부 라이브러리 없이 순수 웹 기술로 구현

## 데이터 모델

```javascript
{
  id:        "고유 ID (타임스탬프 + 랜덤)",
  title:     "일정 제목",
  date:      "YYYY-MM-DD",
  startTime: "HH:MM",       // 선택
  endTime:   "HH:MM",       // 선택
  priority:  "high | medium | low",
  memo:      "메모 내용",    // 선택
  reminder:  "0 | 10 | 30 | 60",  // 분 단위, 선택
  createdAt: "ISO 8601 타임스탬프"
}
```

데이터는 브라우저 LocalStorage에 저장됩니다.
- 일정 데이터: `task-manager-tasks` 키 (JSON 배열)
- 비밀번호: `taskmanager_pw` 키 (해시값)

브라우저 데이터를 삭제하면 일정과 비밀번호가 함께 삭제되므로 주의하세요.
